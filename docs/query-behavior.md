# 질의 특성과 기대 품질

## 목적

어떤 종류의 문장 검색이 잘 되고, 어떤 경우에 정확도가 떨어질 수 있는지 기록한다. 이 문서는 모델 교체나 재정렬 도입 여부를 판단하는 기준으로 사용한다.

## 비교적 잘 되는 질의

다음 유형은 현재 MVP에서도 비교적 기대할 만하다.

- `노란색이 많이 들어간 풍경`
- `겨울 호수에 나무가 많은 장면`
- `바다와 하늘이 같이 보이는 사진`
- `도시 야경`
- `꽃이 많은 정원`

이유는 CLIP 계열 모델이 색감, 전반적인 장면, 분위기, 큰 객체 단위 의미에는 강한 편이기 때문이다.

## 애매하거나 약한 질의

다음 유형은 정확도가 흔들릴 수 있다.

- `닭튀김 부스러기가 있는 사진`
- `책상 위에 아주 작은 나사 하나가 있는 사진`
- `잔디 위에 떨어진 얇은 종이 조각`
- `컵 옆에 묻은 소스 자국`

이유는 현재 방식이 사진 전체의 의미를 강하게 반영하고, 작은 물체나 미세한 상태 변화는 약하게 반영하기 때문이다.

## 한국어 질의에 대한 처리 방식

현재 기본 검색 경로는 다음과 같다.

1. 한국어 문자가 포함된 질의면 다국어 CLIP 텍스트 모델로 직접 임베딩한다.
2. 영어 질의는 기존 CLIP 텍스트 인코더를 그대로 사용한다.

이 방식은 한국어를 번역 없이 바로 처리하는 단순한 기본 경로다.

## 개선 방향

정확도를 높이기 위한 우선순위는 다음과 같다.

1. 다국어 이미지-텍스트 검색 모델의 품질 평가와 교체 검토
2. 더 자세한 캡션 또는 태그 저장
3. 1차 검색 후 후보를 다시 평가하는 재정렬 단계 추가
4. 색상, 날짜, 폴더 같은 부가 필터 추가

## 운영 기준

초기 버전에서는 다음 원칙으로 기대치를 잡는다.

- 장면, 색감, 큰 객체 검색은 우선 지원 대상이다.
- 미세 디테일과 관계 중심 질의는 후속 개선 대상으로 본다.
- 한국어 질의는 다국어 직접 임베딩을 기본으로 사용한다.
- 질의 실패 사례는 문서에 누적해 모델 교체 판단 근거로 남긴다.
