# 결정 0003: RTX 3070 8GB 기준 로컬 GPU 전략

## 상태

채택

## 배경

사용자 PC에 NVIDIA GeForce RTX 3070 8GB GPU가 있다. 로컬 모델로 인덱싱과 검색을 수행할 때, VRAM 제약 안에서 안정적으로 동작해야 한다.

## 결정

초기 기본 전략은 다음과 같다.

- 검색 임베딩은 CLIP을 로컬 GPU에서 실행한다.
- 이미지 설명은 BLIP base를 기본값으로 사용한다.
- 인덱싱 시 GPU 디바이스 정보를 출력한다.
- VRAM 여유가 부족한 경우를 대비해 `--no-caption` 옵션을 제공한다.

## 결과

### 장점

- 검색 핵심 기능을 로컬에서 빠르게 실행할 수 있다.
- 프라이버시를 유지하면서 반복 인덱싱 비용을 줄일 수 있다.
- VRAM이 부족할 때 설명 생성을 끄고 인덱싱을 계속할 수 있다.

### 단점

- 더 큰 캡션 모델은 8GB VRAM에서 부담이 될 수 있다.
- 한국어 검색 품질은 모델 선택에 따라 제한된다.

## 후속 작업

- 배치 크기 조절 옵션 추가
- mixed precision 검토
- 다국어 검색 모델 옵션 추가
