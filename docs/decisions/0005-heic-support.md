# 결정 0005: HEIC/HEIF 파일을 기본 지원

## 상태

채택

## 배경

아이폰과 일부 모바일 기기에서는 사진이 HEIC/HEIF 형식으로 저장되는 경우가 많다. 로컬 사진 검색 도구가 이 형식을 읽지 못하면 실제 사용성이 크게 떨어진다.

## 결정

다음 방식으로 HEIC/HEIF를 지원한다.

- 의존성으로 `pillow-heif`를 추가한다.
- 애플리케이션 시작 시 HEIF opener를 등록한다.
- 인덱싱 대상 확장자에 `.heic`, `.heif`를 포함한다.

## 결과

### 장점

- 아이폰 사진 라이브러리를 그대로 인덱싱할 수 있다.
- 사용자가 사전 변환 작업을 하지 않아도 된다.

### 단점

- 추가 의존성이 생긴다.
- 일부 환경에서는 HEIF 관련 네이티브 라이브러리 설치 상태에 영향을 받을 수 있다.

## 후속 작업

- HEIC 샘플 파일로 실제 인덱싱 테스트 추가
- Live Photo 관련 파생 파일 처리 정책 검토
